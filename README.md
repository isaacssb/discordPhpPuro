# Discord PHP Puro

Uma classe com php para adicionar uma mensagem com requisição

### Pré - Requisitos

Necessário ter PHP instalado na sua maquina
Crie um arquivo credentials.php e adicone constante com o token da api do discord

define('TOKEN', 'tokenAqui');

Lembre de adicionar esse aquivo no .gitignore

### Iniciando projeto

Rode na sua pasta o seguinte comando
php -S localhost:8000 (troque a porta para sua preferência)

Chame a classe conforme o arquivo index.php

Dessa forma você tem uma função do discord sem precisar baixar 40mb de arquivos
